Administration Part

Administration Panel

Fields:

  • Title: the title displayed above the box
  • Username: name of the user to display, or plancast login
  • Password: plancast password (to display friends plans only)
  • Info to display: 2 options:
    • "Specified User Plans" --> display the plan of the specified user
    • "My Friends Plans" --> display the login user friends plans
  • Number of plans to display: specified the number of plan to display, limited to 100, default 10
  • Limit the height to: limit the height of the plans section to the specified number of pixel. A scrollbar would be display id necessary
